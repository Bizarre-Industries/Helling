# Glossary

- **Project**: Incus-scoped tenant boundary used for resource isolation.
- **Template**: Reusable compose or provisioning definition for quick deployment.
- **Workspace**: Contributor local development context.
- **Warning**: System-generated operational risk signal shown in UI.
- **Instance**: Incus VM or LXC workload.
- **Workload**: Runtime unit managed by Incus or Podman.
- **Cluster**: Group of Kubernetes nodes provisioned by Helling.
- **Session**: Authenticated server-side login state.
- **API Token**: Long-lived automation credential, hashed at rest.
- **Internal CA**: Helling-managed certificate authority for per-user Incus trust identities.
- **Recovery Code**: Single-use MFA fallback code.

# v0.1 Scope

## In Scope

- PAM + JWT + TOTP auth model
- Per-user Incus TLS trust identity
- Proxy-first Incus and Podman integration
- SPICE VM console path (Incus VGA-aligned)
- k3s via cloud-init Kubernetes provisioning
- Core Helling API domains (auth, users, schedules, webhooks, kubernetes, system, firewall, audit, events)

## Out of Scope

- LDAP/OIDC/WebAuthn
- Custom role model and advanced ACL authoring
- MicroVM/Cloud Hypervisor runtime support
- CAPN-based Kubernetes management
- BMC management APIs
- Notification channel APIs

## Deferred

- Enterprise auth capabilities (v0.5+)
- CAPN optional path (v0.5+)
- MicroVM integration (post-v0.1)
- Notification channels and alert routing (v0.3)
- BMC integrations (v0.4)
